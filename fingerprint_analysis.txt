### CRITICAL FAILURES (BİZİ ELE VERENLER)

#### 1. 🚨 webgl_vendor = "" (BOŞ)
- **Değer:** `""` (empty string)
- **Normal Chrome:** `"Google Inc."` veya `"NVIDIA"`, `"Intel"`, `"AMD"`
- **Sorun:** `--disable-gpu` flag'i WebGL'i tamamen kapatmış
- **Arkose Tepkisi:** "Missing WebGL data" → şüpheli bot
- **Kanıt:** Roundproxies BDA — missing WebGL = suspicious

#### 2. 🚨 webgl_renderer = "" (BOŞ)
- **Değer:** `""` (empty string)
- **Normal Chrome:** `"ANGLE (NVIDIA GeForce RTX XXXX Direct3D11 vs_5_0 ps_5_0)"`
- **Sorun:** GPU disabled → renderer string boş
- **Arkose Tepkisi:** Yukarıdaki ile aynı

#### 3. 🚨 chrome_runtime_connect = false
- **Değer:** `false` (yok)
- **Normal Chrome:** `true` (var)
- **Sorun:** CDP ile açılan Chrome'da chrome.runtime API'si eksik
- **Arkose Tepkisi:** Puppeteer-extra-stealth bunu patch'lemek zorunda
- **Kanıt:** puppeteer-extra-plugin-stealth — chrome.runtime modülü

#### 4. 🚨 chrome_runtime_sendMessage = false
- **Değer:** `false` (yok)
- **Normal Chrome:** `true` (var)
- **Sorun:** Yukarıdaki ile aynı

### DİĞER GÖZLEMLER

- **timezone_offset = 0**: UTC kullanıyorsun. TR IP ile UTC timezone uyumsuz olabilir ama bu weak signal.
- **canvas_hash = "AABJRU5ErkJggg=="**: Bu çok kısa — muhtemelen 1x1 boş canvas. Gerçek browser'da daha uzun hash çıkar.
- **plugins isimleri**: Hepsi PDF viewer — normal Chrome'da "Native Client" plugin'i de olurdu.

---

### ÖNERİLEN DÜZELTMELER (Öncelik Sırasına Göre)

1. **WebGL Mock**: `--disable-gpu` flag'ini kaldır, ama software renderer kullanmaya zorla
   - `--use-gl=swiftshader` flag'i ekle (Google'ın yazılımsal WebGL implementasyonu)
   - Bu webgl_vendor = "Google Inc." ve webgl_renderer = "SwiftShader" yapacak

2. **chrome.runtime Mock**: CDP ile page'e inject edilecek JS:
   ```javascript
   if (!window.chrome) window.chrome = {};
   window.chrome.runtime = {
     connect: function() { return { onMessage: { addListener: ()=>{} }, postMessage: ()=>{} }; },
     sendMessage: function() {},
     // + diğer required methods
   };
   ```

3. **canvas_hash düzeltme**: WebGL düzeldiğinde canvas hash de düzelecek.

---

### SONUÇ

**Bizi ele veren ana sinyal: WebGL eksikliği.** 
`--disable-gpu` ve `--disable-software-rasterizer` flag'leri WebGL'i tamamen öldürüyor.
Bunu `--use-gl=swiftshader` ile değiştirirsen:
- webgl_vendor = "Google Inc." (veya benzeri)
- webgl_renderer = "SwiftShader" (software renderer)
- canvas_hash = gerçekçi değer

chrome.runtime mock'u da ekleyince 4 kritik sinyalin hepsi düzelecek.
